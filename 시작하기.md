# 썸네일 워크벤치 — 설치 & 실행 가이드

## 1단계: Node.js 설치 (최초 1회)

https://nodejs.org 에서 **LTS 버전** 다운로드 후 설치.
설치 후 Terminal(터미널) 을 열어서 아래 명령어로 확인:

```
node --version
```
`v20.x.x` 같은 버전이 나오면 성공.

---

## 2단계: 이 폴더에서 터미널 열기

Finder에서 이 폴더(`thumbnail-workbench`)를 우클릭 → **폴더로 터미널 열기**
(또는 터미널에서 `cd` 명령으로 이동)

---

## 3단계: 패키지 설치 (최초 1회)

```
npm install
```

FFmpeg, Electron 등 필요한 모든 패키지가 자동으로 설치됩니다.
약 2~3분 소요.

---

## 4단계: 앱 실행

```
npm start
```

앱 창이 뜨면 완료!

---

## 매번 실행할 때 (더블클릭으로 실행)

`썸네일워크벤치_실행.command` 파일을 **더블클릭**하면 바로 실행됩니다!

> **처음 한 번만**: Finder에서 파일을 우클릭 → **열기** → "열기" 버튼 클릭
> (Mac 보안 정책상 최초 1회만 이 절차가 필요합니다. 이후엔 더블클릭만 하면 됩니다.)

또는 터미널에서:
```
npm start
```

---

## 폴더 구조

```
thumbnail-workbench/
├── main.js          ← 앱 메인 (FFmpeg, 파일 저장 담당)
├── preload.js       ← 보안 브릿지
├── package.json     ← 패키지 설정
├── src/
│   └── index.html   ← UI (React)
└── assets/          ← 아이콘 등 (선택)
```

---

## 웹 브라우저에서 열기 (Electron 없이 테스트)

`src/index.html` 파일을 브라우저에서 직접 열어도 동작합니다.
다만 이 경우 FFmpeg 가속이 없고 파일 선택은 기존 방식(드래그 업로드)으로 작동합니다.
